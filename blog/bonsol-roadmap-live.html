ICYMI - We're excited to share our official roadmap outlining the journey ahead for Bonsol! This roadmap represents our commitment to building the most performant, scalable, and decentralized compute platform for Solana.

Phase 1 - Public Beta (Live)

Current Focus: Usability & Onboarding
  • Verifiable compute framework - Core infrastructure for off-chain computation
  • Bonsol CLI - Command-line tools for developers
  • Bonsol SDK - Software development kit for easy integration
  • Documentation & Tutorials - Comprehensive guides for developers
  • PoC development & Use case exploration - AI perceptron models, gaming examples, verifiable finance
  • Hackathons - Community engagement and innovation

Focus: Onboarding experience, comprehensive documentation and tutorials to make it easier to develop with Bonsol. This phase also features proof-of-concept developments, partner integrations, and community engagement through early user feedback. We're welcoming any interested partners to join our ecosystem, the Bonsol Collective, to test our platform!

Phase 2 - Public Alpha (Coming Q1 2026)

Next Focus: Sustainability & Economics
  • Anchor support and Dev Tooling - Enhanced development experience
  • macOS compatibility - Cross-platform support
  • Verifiable functions and Pinocchio support - Additional proof systems
  • Advanced proof explorer - Track the full proof lifecycle
  • Other leading zkVMs integration - Expand beyond RISC Zero
  • Prover Network infrastructure - Cost estimations, workload allocation, incentives and punishment mechanisms
  • Core protocol audit - Security verification

Focus: Building advanced infrastructure for the Prover Network, including its cost estimations, workload allocation, and incentives and punishment mechanisms, supported by feedback from early ecosystem partners. To reinforce security and stability, we will conduct thorough testing of the economics model prior to the next phase.

Phase 3 - Incentivised Testnet & Mainnet (Q3+ 2026)

Future: Full Decentralization & Governance
  • Incentivized testnet - For provers and application builders starting Q3 2026
  • Mainnet launch - With live Prover Network
  • Additional testnet features - Enhanced functionality
  • Token Program Audit + Tokenomics Paper - Economic model transparency
  • Community Governance - Bonsol Foundation establishment
  • TGE + Listing - Token Generation Event and official exchange listings
  • Audit & Bug Bounty - Finalizing core technology with thorough security verification

Focus: This stage will introduce additional testnet features and include both a token program audit and the release of the tokenomics paper. It will also mark the start of community governance through the Bonsol Foundation. To support a smooth Mainnet launch, we will anchor on any feedback gathered during the testnet phase, while audits and a bug bounty program will finalize the core platform.
